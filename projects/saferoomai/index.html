SafeRoomAI – Real-Time Anomaly Detection

Hybrid pipeline integrating object detection and unsupervised anomaly scoring with reproducible data & experiment versioning.

Key Decisions & Timeline

  • Problem: Need real-time anomaly detection on security streams with graceful fallback if RTSP unavailable.
  • Design: YOLO for object events + Autoencoder reconstruction error for anomalies; modular pipeline.
  • Reproducibility: DVC for dataset & model artifacts; MLflow for experiment tracking + params/metrics.
  • Deployment: Packaged with BentoML, containerized and orchestrated (K8s readiness, horizontal scaling).
  • Optimization: Quantization + frame skip & batching target latency reduction (e.g., 120ms → 70ms).

Technical Highlights

  • Fallback Mode: Automatic local file / buffer processing when network feed drops.
  • Version Control: DVC remote ensures deterministic training snapshots; MLflow run IDs anchor metrics.
  • Serving: BentoML API exposing JSON results + latency logging hooks.
  • Monitoring (Planned Upgrade): Export metrics to lightweight dashboard JSON for portfolio metric cards.
